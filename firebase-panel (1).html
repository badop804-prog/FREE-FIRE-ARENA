📖 SETUP GUIDE
  • 1console.firebase.google.com pe jao → naya project banao → Realtime Database enable karo.
  • 2Project Settings → Your Apps → Web App add karo → firebaseConfig copy karo.
  • 3Upar form me sab values paste karo aur "Save & Connect" dabao.
  • 4Done! User Panel aur Admin Panel dono automatically connect ho jayenge.
  • 5Remove karna ho to "Remove" dabao — dono disconnect ho jayenge.
